# ✅ To Do List

A simple terminal-based To Do List app built with Python.

---

## 🛠️ Built With
- Python 3.x
- `os` — for file management

---

## ▶️ How to Run

```bash
python To_Do_List.py
```

---

## ⚙️ Features
- Add new tasks
- View all tasks with numbering
- Mark tasks as completed (removes from list)
- Skip tasks for tomorrow
- Tasks saved in a `.txt` file

---

## 📌 Menu Options
```
1. Add Tasks
2. Show Tasks
3. Completed Tasks
4. Skip for Tomorrow
5. Quit
```
