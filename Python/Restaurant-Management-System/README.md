# 🍽️ Restaurant Management System

## 📌 Description

A command-line based system to manage restaurant orders and calculate the total bill.

---

## 🚀 Features

* Display menu
* Take multiple orders
* Calculate total bill
* User-friendly CLI interface

---

## ▶️ How to Run

```
python Restaurant_Management_System.py
```
