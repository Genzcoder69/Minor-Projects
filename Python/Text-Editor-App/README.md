# 📝 Text Editor App

## 📌 Description

A simple Python-based text editor that allows users to write and save text.

---

## 🚀 Features

* Create and edit text
* Save content to file
* Lightweight and easy to use

---

## ▶️ How to Run

```
python Text_editor_app.py
```
