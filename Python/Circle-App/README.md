# 🔵 Circle App

## 📌 Description

This is a simple Python program that calculates the **area** and **circumference** of a circle based on user input.

---

## 🚀 Features

* Accepts radius as input
* Calculates area of circle
* Calculates circumference of circle

---

## 🧮 Formula Used

* Area = π × r²
* Circumference = 2 × π × r

---

## ▶️ How to Run

```
python Circle1.py
```
