# 💳 QR Payment System

## 📌 Description

A Python project that generates a QR code for making digital payments easily.

---

## 🚀 Features

* Generate QR code for payments
* Easy to scan and share
* Useful for small businesses

---

## ▶️ How to Run

```
python Payments_QrCode.py
```

---

## ⚙️ Requirements

Install required library:

```
pip install qrcode
```

---

## 📂 Project Structure

```
QR-Payment-System/
└── Payments_QrCode.py
```

## 🛠️ Technologies Used

* Python
