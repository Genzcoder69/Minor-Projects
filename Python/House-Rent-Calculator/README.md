# 🏠 House Rent Calculator

A simple terminal-based House Rent Calculator built with Python.

---

## 🛠️ Built With
- Python 3.x

---

## ▶️ How to Run

```bash
python House_Rent_Calculator.py
```

---

## ⚙️ Features
- Calculates electricity bill based on units consumed
- Splits total rent equally among all tenants
- Accepts food and rent expenses
- Remove a tenant who doesn't want to pay

---

## 📌 Inputs Required
- Electricity charge per unit
- Total persons living in house
- Monthly electricity spent (units)
- Monthly house rent
- Food expenses
- Tenant names
